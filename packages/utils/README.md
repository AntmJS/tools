# @antmjs/utils

> 通用工具

### 为什么需要

一些常见的工具的总结，避免每次都要重新造一遍轮子

### 安装

```bash
pnpm add @antmjs/utils
```

### 使用

```js
import {
  debounce,
  throttle,
  version,
  stringify,
  parse,
  decodeParams,
  encodeParams,
  isEqual,
  isString,
  isArray,
  isBoolean,
  isEmptyObject,
  isFunction,
  isNull,
  isNumber,
  isObject,
  isSymbol,
  isUndefined,
  isMobile,
  isIDCard,
  isBankCard,
  isEmail,
  randomNum,
  randomStr,
  formatDigit,
  formatNumber,
  sleep,
  formatDate,
  formatStartUnix,
  formatStartDay,
  filterUndefinedAndNull,
  compareVersion,
  sizeOfStr,
  loopFunc,
  dayjs,
  bigjs,
} from '@antmjs/utils'
```
